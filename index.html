什麼是 HTML？

嘿！你有沒有想過，網頁是怎麼把文字、圖片和影片都放在一起的？答案就是 HTML！它的名字聽起來很專業——「超文本標記語言」——但其實就像是給瀏覽器看的指令，告訴它要把東西放在哪裡。

舉個例子，如果你想要在網頁上寫一句「Hello World」，用 HTML 超簡單，就像這樣：

<h1>Hello World!</h1>

這行指令會在網頁上顯示一個超大的標題，這就是 <h1> 的魔力！HTML 就是這樣，用一些標籤（比如 <p>, <img>）來定義你的內容和格式。

HTML 標籤 🏷️

HTML 的基本單位就是標籤！像 <h1> 是用來顯示標題，<p> 用於段落。每個標籤都是讓網頁鮮活起來的神奇組件！

HTML 屬性 ⚙️

標籤還能帶屬性，像 class 和 id，這就像標籤的「裝飾品」，可以讓你的網頁更有個性化！

HTML + CSS 💅

HTML 是網頁的骨架，但 CSS 才是它的時尚設計師！學會把它們結合起來，打造一個既有內容又漂亮的網站。
